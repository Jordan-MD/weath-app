# 🌦️ Weather App

A responsive weather application built with **HTML**, **CSS**, and **Vanilla JavaScript**. It allows users to search for any city and displays real-time weather information using the OpenWeatherMap API.

---

## 📖 Overview

This project was developed as part of my Frontend learning roadmap. The main objective was not only to consume a REST API, but also to apply software engineering practices such as modular architecture, separation of concerns, asynchronous programming, and clean code.

---

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Display current temperature
* 📍 Display city name
* ☁️ Weather description
* 💧 Humidity
* 👀 Visibility
* 💨 Wind speed
* ⏳ Loading indicator during API requests
* ❌ User-friendly error messages
* 📱 Responsive interface

---

## 🛠️ Technologies

* HTML5

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Navigate into the project

```bash
cd weather-app
```

### Configure your API key

Create or edit the configuration file and add your OpenWeatherMap API key.

```javascript
export const API_KEY = "YOUR_API_KEY";
```

### Run the project

Open `index.html` in your browser.

Alternatively, you can use **Live Server** in Visual Studio Code.
